# Installation


### Step 1: 
Go to todolist(frontend) and sever (backend) directory to install packages
```bash
cd todolist
npm install
```
```bash
cd server
npm install
```
### Step 2: Run the App 
Go to sever and todolist directory and start the server
```bash
cd todolist
npm run dev
```
```bash
cd server
node index.js
```
